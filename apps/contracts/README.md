# Farcaster Risk - Smart Contracts

This directory contains Foundry-based smart contracts for the Farcaster Risk game.

## Planned Features

- Entry fee collection
- Prize pool distribution
- NFT-based game rewards
- Tournament management

## Setup

1. Install Foundry:
```bash
curl -L https://foundry.paradigm.xyz | bash
foundryup
```

2. Install dependencies:
```bash
forge install
```

3. Build contracts:
```bash
forge build
```

4. Run tests:
```bash
forge test
```

## Contracts (Planned)

- `RiskGame.sol` - Main game contract
- `RiskToken.sol` - ERC20 token for entry fees and rewards
- `RiskNFT.sol` - ERC721 NFT for achievements
- `TournamentManager.sol` - Tournament organization and prize distribution
